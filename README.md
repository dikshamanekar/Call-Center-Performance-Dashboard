# 📊 Call Center Data Analysis Dashboard (Excel)

## 📌 Project Overview
This project analyzes a Call Center dataset using Microsoft Excel to evaluate 
customer service performance and agent efficiency.

The dashboard provides insights into call handling, response time, issue 
resolution, and customer satisfaction.

---

## 🛠 Tools Used
- Microsoft Excel
- Data Cleaning
- Pivot Tables
- Pivot Charts
- Excel Dashboard

---

## 📂 Dataset Information
The dataset contains **5000 call center records** with the following fields:

- Call ID
- Agent Name
- Call Date & Time
- Topic of the Call
- Answered (Yes/No)
- Resolved Status
- Speed of Answer (seconds)
- Average Talk Duration
- Customer Satisfaction Rating
- Month

---

## 📊 Key Analysis Performed

### 1️⃣ Agent Performance
- Number of calls handled by each agent
- Resolution rate of agents

### 2️⃣ Call Handling Analysis
- Total calls answered vs unanswered
- Average response time

### 3️⃣ Customer Satisfaction
- Average satisfaction rating
- Satisfaction trends by agent

### 4️⃣ Monthly Call Trends
- Total calls per month
- Topic distribution
